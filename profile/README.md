# рҹ§җ How To Start

### 1) Clone
```bash
git clone <YOUR_REPOSITORY_URL>
cd Truvel
```

### 2) Backend м„Өм • (backend/truvel/.env)
```env
DB_NAME=
DB_USER=
DB_PASSWORD=
DB_PORT=

# docker-composeм—җм„ң мӮ¬мҡ© (web м»Ён…Ңмқҙл„Ҳ лӮҙл¶Җ DB м—°кІ° мЈјмҶҢ)
DB_URL=jdbc:mysql://mysql:3306/${DB_NAME}?useSSL=false&serverTimezone=UTC&allowPublicKeyRetrieval=true

REDIS_HOST=redis
REDIS_PORT=

GOOGLE_API_KEY=

GMAIL_ADDRESS=
GOOGLE_APP_PASSWORD=

JWT_SECRET=
```
<br>

### 3) Backend мӢӨн–ү
#### Docker (к¶ҢмһҘ)
```bash
cd backend/truvel
docker compose up -d --build
```

#### Docker мў…лЈҢ
```bash
docker compose down
```

#### лЎңм»¬ мӢӨн–ү (м„ нғқ)
```bash
cd backend/truvel
# Windows
gradlew.bat bootRun
# macOS / Linux
./gradlew bootRun
```

### 4) Frontend м„Өм • (frontend/.env.local)
```env
NEXT_PUBLIC_API_BASE_URL=http://localhost:8080
NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=
```

### 5) Frontend мӢӨн–ү
```bash
cd frontend
npm install
npm run dev
```

### 6) м ‘мҶҚ
- Frontend: http://localhost:3000
- Backend: http://localhost:8080
